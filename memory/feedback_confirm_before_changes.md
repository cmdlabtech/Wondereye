# Confirm Before Making Changes

## What Happened
User asked "could you create a What's New section" — I immediately implemented the changes. User responded: "I was just asking you. I don't want any changes right now."

## Rule
Treat the following phrasings as **questions or discussion**, not instructions to act:
- "Could you X"
- "What would X look like"
- "Can you X"
- "How would you X"
- "What if we X"

## How to Apply
When a user asks "could you X" or similar, **describe what you'd do and ask for confirmation before making any edits**. Only proceed with changes when the user's intent is unambiguous (e.g. "go ahead", "do it", "make those changes", "implement that").

When in doubt, say what you'd do and ask: "Want me to go ahead with that?"
